## Área destinada para códigos sobre operadores relacionais (>=, ==, <=, !=)

* Maior que ( > )
* Menor que ( < )
* Maior ou igual ( >= )
* Menor ou igual ( <= )
* Igual a ( == )
* Diferente de ( != )

  É importante que você tenha visto a parte de Sistema de Comparação para que fique mais simples o entendimento dos assuntos a seguir
